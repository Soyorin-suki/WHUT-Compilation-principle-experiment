# 编译原理课程实验项目

本项目为武汉理工大学编译原理课程实验项目，采用c++进行编码，旨在实现一个简单的编译器前端。

## 项目状态
已完成部分功能，由于目前已通过课程考核，该项目将不再继续开发和维护。
